Ключевые возможности

Окно поверх всех

Компактное плавающее окно остаётся над другими приложениями. Быстро показывайте/скрывайте без смены контекста.

Переключение по горячей клавише (F12)

Нажмите F12, чтобы мгновенно показать/скрыть оверлей. Оптимально для активной торговли.

Поддержка бирж

Работает с Binance (Futures) и Bybit (Unified). Показывает общий/доступный баланс, нереализованный PnL и открытые позиции.

Интеграция с треем

Меню в системном трее: восстановить окно, переключить оверлей или выйти. Можно скрывать окно из панели задач.

Автоподключение и автозапуск

Автоматически переподключается к бирже и может запускаться вместе с Windows — оверлей готов, когда готовы вы.

Настройки и языки

Выбирайте цвета баланса и PnL (плюс/минус), шрифты и размеры, переключайте язык приложения.

Загрузки

Выберите подходящую сборку установщика. Все варианты — для Windows.

Примечание: некоторые антивирусы могут ругаться на неподписанные файлы. Вы также можете собрать приложение из исходников с помощью скриптов.

Как это работает

  1. Установите и запустите OverlayTrader.
  2. Введите API‑ключ и секрет биржи (Binance или Bybit).
  3. Включите автоподключение, чтобы баланс и позиции всегда были актуальны.
  4. Используйте F12 для показа/скрытия оверлея во время работы или торговли.
  5. Настройте цвета, размеры шрифта и язык под себя.
  6. При желании включите автозапуск вместе с Windows.

Отказ от ответственности: торговля связана с рисками. OverlayTrader не размещает ордера; он лишь отображает данные, полученные по вашему API биржи.
